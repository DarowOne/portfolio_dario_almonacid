Diseño Web · WordPress · HTML · CSS · JavaScript

Diseño y desarrollo sitios web para empresas, emprendedores e instituciones, combinando diseño visual, experiencia de usuario y desarrollo técnico.

Trabajo principalmente con WordPress, pero también realizo personalizaciones mediante HTML, CSS y JavaScript cuando el proyecto requiere soluciones que van más allá de las opciones estándar de un constructor visual.

Mi enfoque es crear sitios rápidos, responsivos y fáciles de administrar, cuidando tanto la estética como el rendimiento y la experiencia del usuario en todos los dispositivos.

He desarrollado proyectos para organismos públicos, empresas, agencias, alojamientos, eventos y negocios de servicios, participando en distintas etapas del proceso: planificación, diseño, implementación, optimización y mantenimiento.

Cada proyecto tiene necesidades diferentes. Por eso busco soluciones prácticas, escalables y orientadas a resultados, evitando plantillas genéricas y adaptando cada sitio a los objetivos reales del cliente.

Generislabs

Optimización técnica y UX

Optimización integral del sitio de Generis Lab enfocada en performance, accesibilidad y experiencia de usuario. Se mejoró la velocidad de carga mediante compresión Gzip y optimización del tiempo de respuesta del servidor. También se aplicó font-display: swap para mejorar la carga de iconografía sin bloquear el render inicial. A nivel de UX y accesibilidad, se corrigieron etiquetas en botones e íconos, se eliminó la restricción de zoom en mobile y se reorganizó la jerarquía de encabezados para una estructura más clara y SEO-friendly. Finalmente, se rediseñó el comportamiento responsive en mobile, mejorando botones, áreas de toque y consistencia visual general.

https://generislabs.com/
